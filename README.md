## How to Use

- Read `theory.md` first for each module
- Run the `.java` files to see output
- Use `interview/` before any Java interview
- Use `cheat-sheet/` as a daily reference

## Prerequisites

- JDK 8+
- Basic Java knowledge (OOP, Collections)

```bash
javac 02-lambda/LambdaBasics.java && java LambdaBasics
```
